## ğŸŸ¢ BASIC GIT COMMANDS

| Action | Command |
| --- | --- |
| Initialize a new repo | `git init` |
| Clone a remote repo | `git clone <url>` |
| Check repo status | `git status` |

| Action | Command |

## ğŸ› ï¸ MODIFYING HISTORY

| Action | Command |
| --- | --- |
